## Overview

Rosters help you organize patients into groups, making it easy to bill for clinic days, facility visits, or recurring patient panels.

## Accessing Rosters

Tap the **Rosters** tab (fourth icon) in the bottom navigation.

## Understanding Rosters

A roster is a collection of patients associated with:
- A specific facility or location
- Default billing codes
- Your billing profile

Rosters are synced with the web app.

## Selecting a Roster

If you have multiple rosters:
1. Tap the roster dropdown at the top
2. Select the roster you want to work with
3. Patient list updates to show that roster

## Creating a New Roster

1. Tap the **+** or **Add Roster** button
2. Enter roster name
3. Select associated facility
4. Optionally add default billing codes
5. Tap **Create**

## Adding Patients

### Search Mode

1. Tap **Add Patient** (person icon in header)
2. Search by name or health card number
3. Select patient from results
4. Patient is added to current roster

### Manual Entry

1. Tap **Add Patient**
2. Select **Manual Entry**
3. Enter patient details:
   - Health card number
   - Version code
   - Name
   - Date of birth
4. Tap **Add**

### Camera Scan

1. Tap **Add Patient**
2. Select **Camera**
3. Scan the health card
4. Verify extracted information
5. Tap **Add**

### Validate Mode

1. Tap **Add Patient**
2. Select **Validate**
3. Enter health card number
4. System validates with OHIP
5. Confirmed patients are added

## Removing Patients

1. Find the patient in the roster
2. Swipe left (iOS) or long-press (Android)
3. Tap **Remove**
4. Confirm removal

<Note>
Removing from a roster doesn't delete the patient's records or claims.
</Note>

## Roster Settings

Tap the settings icon to configure:
- Roster name
- Associated facility
- Default billing codes
- Default diagnostic codes

## Managing Billing Codes

Each roster can have default billing codes:

1. Open roster settings
2. Tap **Billing Codes**
3. Add codes that commonly apply to roster patients
4. These pre-populate when creating claims
