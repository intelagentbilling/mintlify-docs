## Introduction

The IntelAGENT mobile app brings powerful billing capabilities to your iOS or Android device. Perfect for healthcare providers who need to submit claims from clinic, hospital, or on-call settings.

## App Navigation

The app uses a bottom tab navigation with five main sections:

| Tab | Icon | Description |
|-----|------|-------------|
| Claims | File | Create new claims |
| Records | List | View claim history |
| Revenue | Chart | Revenue analytics |
| Rosters | Users | Patient roster management |
| Settings | Gear | Account and app settings |

## Syncing with Web App

The mobile app syncs with the web application:
- Claims submitted on mobile appear on web
- Patient rosters are shared between platforms
- Settings and profile sync automatically

## Offline Capabilities

Limited offline functionality:
- Draft claims saved locally
- Auto-sync when connection restored
- View recently accessed records

## Getting Started

1. [Download and install the app](/docs/mobile-app/installation)
2. Log in with your IntelAGENT credentials
3. Configure your profile in Settings
4. Start submitting claims

## When to Use Mobile vs. Web

| Use Mobile For | Use Web For |
|----------------|-------------|
| Quick single claims | Bulk data entry |
| On-the-go submission | Detailed reports |
| Document scanning | Team management |
| Revenue check | Import/export |
| Roster patient visits | Complex scenarios |
