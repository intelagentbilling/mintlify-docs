## Introduction

The IntelAGENT web application provides a comprehensive platform for managing your medical billing workflow. Access it at [app.intelagent.ca](https://app.intelagent.ca).

## Main Features

### Dashboard
Your central hub for monitoring billing performance with real-time revenue analytics, claim summaries, and quick access to recent activity.

### Claims Management
Create, edit, and track claims with:
- New claim submission with built-in validation
- Claim history with advanced filtering
- Calendar view for visual scheduling
- Bulk operations for efficiency

### Patient & Billing Tools
- **Patient List** - Manage patient rosters and associations
- **Billing Sheet** - Spreadsheet-style billing entry
- **Import** - Bulk upload claims via CSV/Excel

### Reports
Access detailed financial reports:
- Remittance advices from the Ministry
- Service charges and invoices
- Excel exports for accounting

### Advanced Features
- **Scenarios** - Compare different billing configurations
- **Team Access** - Delegate billing to team members with role-based permissions

## Navigation

The web app uses a sidebar navigation with the following main sections:

| Section | Description |
|---------|-------------|
| Dashboard | Revenue overview and quick stats |
| Claims | View and manage all claims |
| New Claim | Create a new claim |
| Calendar | Visual claim scheduling |
| Patient List | Manage patient rosters |
| Billing Sheet | Spreadsheet billing entry |
| Import | Bulk data upload |
| Reports | Financial reports |
| Scenarios | Billing comparisons |
| Settings | Account and team management |

## Browser Support

IntelAGENT works best with modern browsers:
- Chrome (recommended)
- Firefox
- Safari
- Edge
