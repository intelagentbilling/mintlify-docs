## Claims List

The Claims page displays all your claims with powerful filtering and search capabilities.

## Viewing Claims

Navigate to **Claims** in the sidebar to see your claim history. Each claim shows:
- Patient name
- Service date
- Billing codes
- Claimed amount
- Status

## Filtering Claims

### By Status

Filter claims by their current status:

| Status | Description |
|--------|-------------|
| Draft | Saved but not submitted |
| Submitted | Sent to OHIP, awaiting processing |
| Partial | Some services paid, others pending/rejected |
| Successful | Fully paid |
| Rejected | Denied by OHIP, requires attention |

### By Date Range

1. Click the date filter
2. Select a preset range (Today, This Week, This Month) or custom dates
3. Claims within that service date range are displayed

### By Patient

Search for claims by patient name or health card number using the search bar.

## Editing Claims

### Draft Claims

Draft claims can be fully edited:
1. Click on the claim to open it
2. Modify any fields
3. Click **Save** or **Submit**

### Submitted Claims

For claims already submitted:
1. Click on the claim to view details
2. Depending on status, you may be able to edit certain fields
3. Rejected claims can be corrected and resubmitted

## Claim Actions

### Duplicate

Create a copy of an existing claim:
1. Open the claim
2. Click **Duplicate**
3. Modify the service date and any other details
4. Submit the new claim

### Delete

Remove a draft claim:
1. Open the claim
2. Click **Delete**
3. Confirm deletion

<Warning>
Submitted claims cannot be deleted. Contact support if you need to cancel a submitted claim.
</Warning>

## Bulk Operations

Select multiple claims using checkboxes to:
- Submit multiple drafts at once
- Export selected claims
- Apply bulk edits

## Pagination

Large claim lists are paginated. Use the navigation controls to browse through pages or adjust the number of items displayed per page.
